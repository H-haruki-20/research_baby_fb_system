# 乳幼児の生体情報フィードバックシステム
本プロジェクトは、乳幼児の心拍や呼吸などの生体情報を用いて、触れ合い体験を向上させるシステムを開発することを目的としています。このシステムを通じて、乳幼児との触れ合いによる親和性の向上や、心理的影響を研究します。

## システム概要
このプロジェクトでは以下を利用しています：
- **ハードウェア:**
  - Arduino Uno
  - 心拍/振動モジュール
  - エアポンプと吸音材
- **ソフトウェア:**
  - Arduino IDE (C++)
  - Python (データ分析用)
  - Git (バージョン管理)

システムの主な機能：
1. **心拍のフィードバック**: 手首バンドで乳幼児の鼓動を再現。
2. **呼吸の再現**: 人形の腹部の膨張/収縮。
3. **体温の再現**: 温度センサーとヒーターを使用予定。
